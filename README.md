# ECOM247 — Pesquisa de Mercado ML

Plataforma premium de pesquisa e análise de produtos para o Mercado Livre.

## Stack

- React 18 + Vite
- Tailwind CSS
- Recharts (gráficos)
- React Router v6
- Lucide React (ícones)

## Funcionalidades

- Landing page com branding ECOM247
- Login e cadastro
- Dashboard com:
  - Busca por texto, imagem ou link ML
  - Score de oportunidade
  - Análise de concorrentes
  - Gráfico de demanda
  - Calculadora de precificação
  - Histórico de pesquisas
  - Export para PDF

## Como rodar localmente

```bash
npm install
npm run dev
```
